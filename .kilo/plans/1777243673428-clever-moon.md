# Earthy Design System Integration Plan

## Overview

Apply the TechtonicCMS earthy design system ŌĆö color palette, volcanic rock textures, and geological accents ŌĆö to the existing SvelteKit + shadcn-svelte dashboard.

## Key Changes

### 1. Color Palette Overhaul (`src/routes/layout.css`)

Replace the current OKLCH-based shadcn theme with the earthy geological palette.

**New CSS custom properties to add:**

- `--bg-page`, `--bg-surface`, `--bg-base`, `--bg-surface-hover`
- `--text-primary`, `--text-secondary`
- `--border-subtle`, `--border-focus`
- `--accent-copper` (#C75A1A), `--accent-moss` (#3A5A40), `--accent-gold` (#D4A017)
- `--rock-crack` (adapts per mode)

**Light mode:**

- Page bg: `#F5F0E8`, Card: `#FFFFFF`, Hover: `#F0EBE0`
- Text: `#1A1A1E`, Secondary: `#6B6B75`, Border: `#E0DCD4`
- Rock crack: `rgba(26, 26, 30, 0.055)`

**Dark mode:**

- Page bg: `#0D0D0F`, Card: `#16161A`, Hover: `#1E1E24`
- Text: `#F2EDE4`, Secondary: `#8C8C96`, Border: `#2A2A32`
- Rock crack: `rgba(242, 237, 228, 0.055)`

**Map shadcn variables:** Update `--sidebar-*`, `--primary`, `--secondary`, `--muted`, `--accent`, `--card`, `--background`, `--foreground`, `--border`, `--ring` to use the earthy hex values so all existing shadcn components inherit the theme.

### 2. Typography Update

- Replace `@fontsource-variable/roboto` with `@fontsource-variable/inter`
- Add `@fontsource/jetbrains-mono` (400, 500)
- Update `--font-sans` to Inter, add `--font-mono` for JetBrains Mono
- Apply `font-mono` to timestamps/IDs where appropriate (future pages)

### 3. Rock Texture Component (`src/lib/components/rock-texture.svelte`)

Create a new Svelte component porting the React `RockTexture.tsx` pattern:

- Inline SVG with `<pattern>` containing fracture polylines/paths
- Stroke color bound to `var(--rock-crack)` CSS custom property
- `preserveAspectRatio="none"` with `absolute inset-0 pointer-events-none`
- Use the dark SVG paths from the reference (the sharper, more angular variant)

### 4. Sidebar Styling (`src/lib/components/app-sidebar.svelte`)

- Add `<RockTexture />` overlay inside the sidebar root
- Update logo block: copper (`#C75A1A`) square background with "T" mark (or Globe icon), "Techtonic" text
- Update active nav state:
  - 3px copper left-border accent (`border-l-[3px] border-l-[var(--accent-copper)]`)
  - Faint copper-tinted background (`bg-[var(--accent-copper)]/5`)
  - Icon and text switch to copper color
- Ensure the sidebar background uses `--bg-surface` so texture is visible
- Move `ModeToggle` from the header into the sidebar footer (below nav, above user profile)
- Update `app-nav-user.svelte` avatar fallback to use moss green (`--accent-moss`) background

### 5. Top Bar / Header Styling (`src/routes/(app)/+layout.svelte`)

- Add `<RockTexture />` overlay to the `<header>` element
- Update header background to `--bg-surface`
- Add `border-b border-[var(--border-subtle)]`
- Keep existing `Sidebar.Trigger`, separator, and right-side actions
- The breadcrumb area is currently minimal; we can enhance it later if needed

### 6. Mode Toggle Update (`src/lib/components/mode-toggle.svelte`)

- Change from icon-only `Button` to a full-width sidebar-style row button
- Show Sun/Moon icon + "Light Mode" / "Dark Mode" text
- Use sidebar menu button styling or a custom row with hover bg

### 7. Static Assets

- Copy `rock-texture-light.svg` and `rock-texture-dark.svg` to `static/` for potential future direct usage (optional, since we're inlining)
